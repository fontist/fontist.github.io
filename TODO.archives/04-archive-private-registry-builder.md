# 04 — RegistryBuilder: fonts.json + font-metadata.json

## Why

fontist.org's `fetch-data.sh:96-100` expects two registries at the archive
root:

- `fonts.json` — FontsRegistry: `canonical_name → formula slugs`
- `font-metadata.json` — FontMetadataFile: per-face metadata

Today, archive-private produces neither. The old `fontist/fontist-archive`
repo has them, generated by `formulas/process/generate_font_metadata.rb`.
For the new pipeline, archive-private must own this generation.

## Source of truth

The formula YAML is canonical. RegistryBuilder walks every formula YAML +
each face's audit output to assemble both registries.

## fonts.json shape (FontsRegistry — domain.ts:96)

```json
{
  "generated_at": "2026-06-29T12:00:00Z",
  "total_fonts": 5963,
  "total_formulas": 4283,
  "fonts": [
    {
      "canonical_name": "ABeeZee",
      "slug": "abeezee",
      "formulas": ["google/abeezee"],
      "style_count": 2
    }
  ]
}
```

Built by:
1. Walk `formulas/Formulas/**/*.yml`
2. For each formula, read `fonts[].name` + count `fonts[].styles[]`
3. Group by canonical family name (case-sensitive)
4. Aggregate `formulas[]` slugs per family

## font-metadata.json shape (FontMetadataFile — domain.ts:103)

```json
{
  "generated_at": "2026-06-29T12:00:00Z",
  "total_fonts": 50,
  "redistributable": 50,
  "fonts": [
    {
      "slug": "abeezee",
      "formula_path": "Formulas/google/abeezee.yml",
      "redistributable": true,
      "primary_family": "ABeeZee",
      "coverage_file": "coverage/google/abeezee/ABeeZee-Regular.json",
      "woff_file": "woff/google/abeezee/ABeeZee-Regular.woff"
    }
  ]
}
```

**Field rename:** `woff2_file` → `woff_file` (per WOFF decision, plan 01).

Built by:
1. Walk every formula YAML
2. For each face in `fonts[].styles[]` and `font_collections[].fonts[].styles[]`:
   - slug = formula slug
   - formula_path = path relative to formulas root
   - redistributable = formula's `spdx_license` is in `REDISTRIBUTABLE_PREFIXES`
   - primary_family = formula's `fonts[0].name`
   - coverage_file = `coverage/{slug}/{PSName}.json` if exists
   - woff_file = `woff/{slug}/{PSName}.woff` if redistributable

**Coverage-file / woff-file existence checks** are important: if a face
failed to audit, omit that entry (or mark coverage_file/woff_file as null).

## RegistryBuilder responsibility

```ruby
module FontistArchivePrivate
  class RegistryBuilder
    def initialize(formula_dir:, archive_root:)
      @formula_dir = formula_dir
      @archive_root = archive_root
    end

    def call
      write_fonts_json
      write_font_metadata_json
    end

    private

    def collect_families; end
    def collect_face_entries; end
    def write_fonts_json; end
    def write_font_metadata_json; end
  end
end
```

## Builder wiring

After the per-formula loop in `Builder#call`, before exiting:

```ruby
RegistryBuilder.new(formula_dir: @formula_dir, archive_root: Dir.pwd).call
```

So registries reflect the current state after each batch.
