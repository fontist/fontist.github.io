## Fetch Script: `scripts/fetch-data.sh`
```bash
#!/bin/bash
# Fetches latest data from fontist-archive and formulas
set -e

ARCHIVE_URL="https://raw.githubusercontent.com/fontist/fontist-archive/main"
FORMULAS_URL="https://raw.githubusercontent.com/fontist/formulas/main/docs/public"

# Coverage + specimens from fontist-archive
echo "Fetching coverage data..."
mkdir -p public/coverage
curl -sL "$ARCHIVE_URL/coverage/" | ... # or use git archive

echo "Fetching font specimens..."
mkdir -p public/fonts
# ... fetch woff2 files

echo "Fetching font registry..."
curl -sL "$ARCHIVE_URL/fonts.json" > public/fonts.json
curl -sL "$ARCHIVE_URL/font-metadata.json" > public/font-metadata.json

# Formula data from formulas
echo "Fetching formulas data..."
curl -sL "$FORMULAS_URL/formulas-data.json" > public/formulas-data.json
```
