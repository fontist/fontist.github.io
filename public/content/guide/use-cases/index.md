# Use Cases

Fontist formulas can be used in many different scenarios. Choose the use case that matches your needs:

## By Environment

| Use Case | Description | Key Considerations |
|----------|-------------|-------------------|
| [Desktop Publishing](/formulas/guide/use-cases/desktop-publishing) | InDesign, Scribus. Illustrator | License compliance. font availability |
| [Web Development](/formulas/guide/use-cases/web-development) | Self-hosted web fonts | WOFF/WOFF2 conversion. subsetting |
| [Document Generation](/formulas/guide/use-cases/document-generation) | Pandoc. Asciidoctor. LaTeX | Reproducible builds. CI integration |
| [PDF Generation](/formulas/guide/use-cases/pdf-generation) | Prawn. wkhtmltopdf. Prince | Font embedding. rendering quality |
| [CI/CD & Servers](/formulas/guide/use-cases/server-side) | GitHub Actions. GitLab CI | Caching. license compliance |
| [Docker & Containers](/formulas/guide/use-cases/containerized) | Docker images. Kubernetes | Image size. font persistence |
| [Cloud VMs](/formulas/guide/use-cases/cloud-vms) | AWS. Azure. GCP | Platform licensing. scalability |

## By License Type

| License Category | Use Case Fit | Examples |
|------------------|--------------|----------|
| [Open Source](/formulas/licenses/ofl) | Any use case | OFL. Apache. MIT. CC0 |
| [Freely Distributable](/formulas/licenses/microsoft-web) | Most use cases | Microsoft Web Fonts. Freeware |
| [Platform Restricted](/formulas/licenses/apple-only) | Platform-specific development | macOS system fonts. iOS apps |
| [Bundled Software](/formulas/licenses/ms-office) | With parent software installed | MS Office fonts. Adobe fonts |
